# @runnablejs/app

React Router (Remix) UI for Runnable. Used internally by `@runnablejs/express` and `@runnablejs/hono`.

## Usage

This package is not meant to be used directly. Use `@runnablejs/express` or `@runnablejs/hono` instead.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
